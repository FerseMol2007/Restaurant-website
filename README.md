# 🍽️ Mom's Kitchen - Restaurant Website

## 📌 About Project
Mom's Kitchen is a simple restaurant website created using HTML and CSS. 
The website displays popular food items with descriptions and provides a contact form for customers.

## 🛠️ Technologies Used
- HTML5
- CSS3

## ✨ Features
- Restaurant homepage design
- Navigation menu (Home, Recipe, Contact)
- Food recipe cards with images and descriptions
- Contact form section
- Simple and clean layout
